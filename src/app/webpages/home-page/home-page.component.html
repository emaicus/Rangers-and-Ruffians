Rangers and Ruffians

Version 3.0.0: Tenth Anniversary Edition RC 1

10th Anniversary Edition: Release Candidate 1

Ten years! This new edition of Rangers and Ruffians aims to bring things back to the basics and get us to a place where the game is easy and fun to play. The rule of the day is simplification, and that means removing as many sources of complication as possible while focusing on the things that make the game great. Among other things, 10th Anniversary RC1 has the following changes:
  1. Simplified Combat Roll to hit is removed, offhand actions return, and called shots make their debut. Changes to combat should make the game feel fast, tactical, and epic!
  2. Simplified Character Options False choice was creating design bloat, making the game harder to play and run. We've stripped the game back to ten races and eight classes, which are honed to their core features. While removing options like skill trees decreases customization, it means that balance is easy and building a character should take minutes while still resulting in years of fun.
  3. So much more! Read the full rules document to see what else changed. From Death's Door Changes to Called Shots, less healing to simplified weapons, this edition gets back to that original core of Rangers and Ruffians!
