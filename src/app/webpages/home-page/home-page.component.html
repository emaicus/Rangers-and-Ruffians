Version 3.0.0 Release Candidate 3

Release Candidate 3

This version of Rangers and Ruffians is a release candidate! That means it's almost ready for showtime. The site will be updated frequently in the coming weeks with new rules and features as the documentation is completed. Check back often for updates! Release candidate 3 includes the following changes:
  1. Removal of "Roll to Hit" Accuracy checks have been removed from the game to increase the speed and fun of combat!
  2. Addition of Deductive Armor Armor now reduces damage taken as the result of an enemy attack.
  3. The return of "Offhand Actions" On each turn, characters may now take an "Action" and an "Offhand Action." Actions typically deal damage or perform some other greater feat. Offhand actions are usually tactical, such as buffing an ally, debuffing an enemy, or healing.
